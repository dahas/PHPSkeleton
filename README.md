# PHP Application Skeleton

Everything you need to build a PHP App from scratch.

## Installation

````
$ git clone https://github.com/dahas/PHPSkeleton.git
$ cd PHPSkeleton
$ cd application
$ composer install
````

## Running tests

````
$ cd application
$ composer test
````

## Add template parser

````
$ cd application
$ composer require dahas/easy-template
````
